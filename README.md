# ⚡ JavaScript Práticas

Repositório criado para armazenar exercícios e práticas desenvolvidos durante meus estudos de JavaScript, com foco no fortalecimento dos fundamentos da linguagem e na aplicação de conceitos essenciais do desenvolvimento web.

## 🚀 Sobre

Este repositório reúne exemplos, exercícios e testes práticos realizados ao longo do aprendizado de JavaScript.

Os conteúdos estão organizados por temas, permitindo acompanhar a evolução dos estudos desde conceitos básicos até tópicos mais avançados da linguagem.

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
